# Aero SMP ✈️

This is the official website for the **Aero SMP** project, a Minecraft server based on the Create Aeronautics mod

## My Role (Frontend)
I am the author of the entire source code for the project (both client and server sides), with a primary focus on **Frontend development**

What I implemented:
* **UI/UX and Layout (Main Focus):** Designed and built the interface from scratch. Configured responsive Flexbox grids for mobile devices
* **Styling:** Implemented a Glassmorphism effect, floating CSS animations, and smooth micro-interactions
* **Functionality:** Set up frontend logic, modal windows, and direct file distribution for the client modpack (`.mrpack`)
* **Backend:** Wrote the data processing logic in Python (with the help of my partner).

*(My partner assisted the website on the DevOps side, handling hosting configuration, server folder structure, and deploying my code)*

## Tech Stack
* **Frontend:** HTML5, CSS3, Vanilla JS
* **Backend:** Python
